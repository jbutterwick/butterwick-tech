Prairie Zola Template

2026-01-11 ◦ 1 min ◦
  • software
◦ living

prairie

A couple of months ago I got interested in Zola, a static site generator built with rust.

I did some experimenting with it and considered using it for a personal project I had been working on at the time, exploring the various themes the zola community has to offer. After much deliberation, I settled on Papaya, a great theme that I felt met my need for a smart, simple UI. The creator of papaya, Justin Tennant, mentions in his writeup on papaya that he was interested in using the Lunaria Design color palletes but strayed away from it. Prairie started out as basically that: Papaya in Lunaria colors.

Eventually, the project I had started prairie for was abandoned. Seeing prairie as being a separate project of its own, I decided to keep it around, thinking I might use it again on some other project. As a living project
